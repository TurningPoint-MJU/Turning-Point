# 공모전 제출용 기획서

## 기획 배경

### 문제 정의

현재 K리그 팬들은 경기 결과를 이해하는 데 어려움을 겪고 있습니다:

- **"점유율은 높은데 왜 졌지?"** - 단순 통계만으로는 경기 흐름을 파악하기 어려움
- **"후반에 갑자기 밀린 이유가 뭐지?"** - 골이 없어도 경기가 기울어지는 순간을 설명할 수 없음
- **"골 말고 뭐가 결정적이었지?"** - 골 중심의 하이라이트만으로는 경기 전체 흐름을 이해할 수 없음

### 기존 솔루션의 한계

- **경기 기록 나열**: 단순 통계 수치만 제공
- **골 중심 하이라이트**: 실제 경기 흐름 변화를 놓침
- **전문 용어 과다**: 일반 팬이 이해하기 어려운 분석

## 제안 솔루션

### 핵심 가치 제안

> **"이 경기는 언제부터 기울었는가?"**
> 
> AI가 찾아낸 K리그 경기 흐름의 변곡점

### 차별성

#### 1. 흐름 중심 분석
- 단순 통계가 아닌 **경기 흐름의 변화**에 초점
- 5분 단위 모멘텀 점수로 경기 주도권 변화를 시각화

#### 2. 골 없는 변곡점도 설명
- 골이 없어도 경기가 기울어지는 순간을 탐지
- 슈팅, 패스, 수비 라인 변화 등 다양한 지표 종합 분석

#### 3. 팬 눈높이 설명
- 전술 용어 최소화
- "아 그래서 졌구나"를 느낄 수 있는 자연어 설명

### 기술적 차별성

- **설명 가능한 AI (Explainable AI)**: Rule 기반 알고리즘으로 모든 판단 근거를 설명 가능
- **실시간 분석 가능**: 트래킹 데이터 없이도 기본 이벤트 데이터만으로 분석
- **확장 가능한 구조**: 향후 LLM 연동으로 더 자연스러운 설명 생성 가능

## MVP 기능

### STEP 1. 경기 흐름 수치화
- 5분 단위 팀별 경기 주도권 점수 계산
- 타임라인 그래프로 시각화

### STEP 2. 변곡점 자동 탐지
다음 4가지 지표 중 2개 이상 충족 시 변곡점 확정:
- 슈팅/xG 급증 또는 급감
- 공격 지역 점유 변화
- 수비 이벤트 평균 위치 변화
- 패스 성공/실패 패턴 변화

### STEP 3. 팬 친화형 설명 생성
- 변곡점 발생 시각
- 주요 변화 지표
- AI 생성 설명 문장
- 관련 이벤트 목록

## 성공 기준

- 팬이 결과를 보고 **"아, 이때부터 졌구나"**라고 말하면 성공
- 축구 지식 없는 사람도 이해 가능한 설명
- 실제 K리그 경기 데이터로 검증

## 향후 발전 방향

1. **LLM 연동**: 더 자연스러운 설명 생성
2. **실시간 분석**: 경기 중 실시간 변곡점 탐지
3. **팀별 특성 학습**: 팀별 플레이 스타일 반영
4. **예측 기능**: 변곡점 이후 경기 결과 예측

## 기술 스택

- **Backend**: Python, FastAPI
- **Data Analysis**: Pandas, NumPy
- **Visualization**: Matplotlib, Plotly
- **AI/ML**: Rule-based (향후 LLM 확장)
